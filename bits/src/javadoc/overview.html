Bits API

This library builds on the API of java.util.BitSet in a variety of ways:

  • Provides a read-only interface to a BitSet, so it is possible to expose it via your API without exposing mutators
  • Provides convenience methods taking IntConsumer and similar for iterating contents
  • Allows for bit sets which contain more than Integer.MAX_VALUE bits
  • Provides implementations of large bit sets over
    • Standard Java-long arrays
    • Off-heap memory provided by sun.misc.Unsafe or equivalent
    • Memory mapped files

Use the factory methods on Bits and MutableBits to obtain instances. For most purposes, it should be possible to use an instanceof Bits as a drop-in replacement for BitSet.
